## 📝 使用说明

### 开发者使用说明

1. **导入项目**
   ```
   打开微信开发者工具
   → 导入项目
   → 选择 HNYC 目录
   → 填入 AppID
   ```

2. **配置环境**
   ```
   修改 project.config.json (AppID)
   修改 app.js (云环境 ID)
   ```

3. **部署云函数**
   ```
   右键 cloudfunctions/login
   → 上传并部署：云端安装依赖
   ```

4. **创建数据库**
   ```
   云开发控制台
   → 创建集合
   → 设置索引
   → 配置权限
   ```

5. **运行项目**
   ```
   点击编译按钮
   → 查看预览效果
   ```

### 终端用户使用说明

1. **首次使用**
   - 自动登录
   - 创建默认账号

2. **添加植宠**
   - 我的空间 → 添加植宠
   - 填写信息 → 保存

3. **创建任务**
   - 植宠详情 → 添加任务
   - 设置提醒 → 保存

4. **发布动态**
   - 公园广场 → 发布
   - 输入内容 → 发布
