# 花鸟鱼虫小程序

一款面向花鸟鱼虫爱好者的微信小程序，为日常喜爱养花养草、饲养鱼龟鸟类等小宠物的人群提供全面的管理和社交功能。

## 功能特性

### 1. 我的空间

#### 我的植宠
- ✅ 添加、编辑、删除各类植物和小宠物信息
- ✅ 支持多种类型：植物（花卉、树木、多肉）、动物（鱼类、龟类、鸟类、昆虫）
- ✅ 完整的个人饲养档案库
- ✅ 封面图片和相册管理
- ✅ 自定义提醒设置

#### 定时任务
- ✅ 创建浇水、修剪、喂食、晒背等个性化任务
- ✅ 关联特定植物或宠物对象
- ✅ 设置重复周期（每天、每周、自定义）
- ✅ 提醒时间和优先级设置
- ✅ 任务完成自动发布到公园广场

#### 对象主页
- ✅ 专属档案页面展示
- ✅ 年龄、初见日期、陪伴时长等基础信息
- ✅ 双时间线展示：
  - 成长时间线：展示对象的生长和照看记录
  - 任务时间线：展示任务清单及完成情况

### 2. 公园广场

#### 动态内容展示
- ✅ 类似微信朋友圈的排版布局
- ✅ 小红书风格的图文展示形式
- ✅ 内容分类筛选（植物、花卉、鱼类、龟类、鸟类等）
- ✅ 最新发布和热门推荐排序

#### 社交互动
- ✅ 访问其他用户个人主页
- ✅ 查看他人饲养的植宠列表及动态
- ✅ 直接访问特定植宠对象主页
- ✅ 关注/取消关注功能
- ✅ 点赞、评论互动

- ✅ 绿色、蓝色等自然色系主题

## 技术架构

### 前端技术栈
- **框架**: 微信小程序原生框架

### 数据库设计

#### 集合说明

1. **users** - 用户信息
   - 基本信息：昵称、头像、性别、简介
   - 社交关系：关注列表、粉丝列表

2. **pets** - 植宠信息
   - 基本信息：名称、类型、子类型、描述
   - 图片：封面图、相册
   - 时间：生日、初见日期
   - 提醒：自定义提醒设置

3. **tasks** - 任务信息
   - 任务内容：标题、类型、描述、优先级
   - 关联对象：植宠 ID
   - 时间设置：提醒时间、重复周期
   - 状态：是否完成、完成时间

4. **posts** - 动态信息
   - 内容：文字、图片、视频
   - 关联：用户、植宠
   - 互动：点赞、评论
   - 类型：日常、任务、成就

## 快速开始

### 1. 环境准备
- 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
- 注册微信小程序账号，获取 AppID

### 2. 配置项目
1. 打开微信开发者工具
2. 导入项目，选择项目目录
3. 在 `project.config.json` 中填入你的 AppID
4. 在 `app.js` 中配置云开发环境 ID

### 3. 云开发配置
1. 开通微信云开发
2. 创建云环境
3. 创建数据库集合：
   - users
   - pets
   - tasks
   - posts
   - comments
4. 上传云函数：
   - 右键 `cloudfunctions/login` 目录
   - 选择"上传并部署：云端安装依赖"

### 4. 数据库索引
为以下字段创建索引以提升查询性能：
- pets: _openid, type, isPublic, createdAt
- tasks: _openid, petId, isCompleted, nextExecuteTime
- posts: _openid, userId, petId, createdAt

### 5. 运行项目
- 点击编译按钮即可预览
- 使用真机预览功能在手机上测试

## 使用说明

### 添加植宠
1. 进入"我的空间"页面
2. 点击"添加植宠"按钮
3. 填写植宠信息（名称、类型、图片等）
4. 设置提醒（可选）
5. 点击"保存"

### 创建任务
1. 进入"我的空间"或"任务管理"页面
2. 点击"创建任务"
3. 选择关联植宠
4. 设置任务类型、时间、优先级
5. 选择是否公开（公开后完成时会发布到广场）
6. 点击"保存"

### 发布动态
1. 点击底部导航栏"公园广场"
2. 点击右上角"发布"按钮
3. 选择关联植宠（可选）
4. 输入内容、上传图片
5. 选择动态类型
6. 点击"发布"
